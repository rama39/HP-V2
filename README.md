# HP-V2
Máquina de gerenciamento de HP para Dungeons and Dragons (MGHPRPG). Segunda versão

O diferencial com a V1 é a portabilidade, incluindo baterias e uma placa de circuito para os componentes.

## Funcionalidades
- Permite somar e subtrair Pontos de Vida (HP)
- Considera danos superiores à vida*2
- Computa testes contra morte, incluindo 1 natural e 20 natural

## Diferencial V2
- Inclui esquemático e PCB feitos no Kicad
- Inclui baterias recarregáveis
- Processador esp32 no lugar do arduino atmega 2560
